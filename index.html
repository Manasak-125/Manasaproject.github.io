This article goes into great detail on the insights that were found from analyzing sales data from superstore sales data.

Tools Used:

  • Kaggle :Data Source
  • Ms-Excel :Data exploration and Data Cleaning

Ask

The first phase of the data analysis journey begins with comprehending the problem statement at hand.

process

Analyze

The analysis phase is dedicated to extracting pertinent information and insights that are crucial for addressing the business issue.

HR Data Analytics - Dashboard

I am excited to present my recent project where I have created an interactive dashboard using Power BI, utilizing an HR dataset sourced from Kaggle. This project involved a comprehensive workflow encompassing data collection, aggregation, preparation, and visualization.

The dataset provided consisted of three separate datasheets, each representing attendance data for different months. With a keen focus on exploring attendance patterns and trends, I meticulously analyzed each datasheet.

Harnessing the functionalities of Power BI, I employed a wide array of functions and DAX formulas to create essential measures and metrics. These measures played a crucial role in shaping the dashboard's insights and user experience.

Through this project, I aimed not only to demonstrate proficiency in data analysis and visualization but also to provide actionable insights to stakeholders. By leveraging HR data, I aimed to offer valuable insights that can inform strategic decision-making within organizations.

This project represents my dedication to utilizing advanced analytical tools and techniques to derive meaningful insights from complex datasets, ultimately contributing to informed decision-making and organizational success.

Tools Used:

  • Kaggle :Data Source
  • Power BI :Data aggregation,Date Cleaning,Date preparation,Date Visualization
